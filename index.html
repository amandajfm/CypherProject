Mensagens Criptografadas

Que tal cifrar uma mensagem e torná-la secreta, para que só você e quem você escolher possa saber seu significado?

Primeiro, preciso que você escolha o número de deslocamento para a nossa cifra.
Isso vai garantir que só quem saiba esse número possa decifrar sua mensagem.
Depois, digite a frase a ser cifrada e clique no botão. Vamos lá?

Criptografe

Veio aqui para decifrar alguma mensagem? Então este aqui é seu campo.
Mas não se esqueça, você deve escolher o mesmo número de deslocamento que foi usado para cifrar essa mensagem.
